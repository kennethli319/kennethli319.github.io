What this lens can—and cannot—say

Rank measures readability. It asks where the realized output sits in a fitted vocabulary-wide readout at a layer.

Rank is not calibrated confidence. A strong intermediate rank does not prove conscious consideration, causation, or correctness.

Errors remain evidence. Generated paths, reference mismatches, token boundaries, and held-out status stay visible in each report.
